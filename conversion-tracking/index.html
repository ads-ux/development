Conversion Tracking API

The Vrume Conversion Tracking API enables advertisers to track conversions, monitor performance, and optimize campaigns using dynamic tokens and server-to-server (S2S) postbacks. Proper tracking ensures accurate reporting, clean attribution, and scalable growth.

Integration Guide

How Server-to-Server Tracking Works

Server-to-server tracking eliminates browser-based tracking issues by sending conversion data directly between servers. This increases reliability, improves attribution accuracy, and reduces data loss caused by ad blockers or browser restrictions.

1

Create a Conversion Goal in the Dashboard

Log in to your advertiser panel and configure your conversion goals:

Configure Conversion Goals â†’
2

Append the Tracking Token to Your Landing Page

When a user clicks your ad, a unique tracking ID is generated and passed to your page.

Landing Page URL 
https://yourdomain.com/?tracker={conversions_tracking}
3

Store the Unique Conversion ID

Save the token in a session variable, cookie, or database so it can be used when the conversion occurs.

4

Send the Postback Request

Fire the postback URL when a conversion event occurs on your server:

S2S Postback URL 
https://YOUR_TRACKING_DOMAIN/tag.php?goal=GOAL_ID&tag={conversions_tracking}&value={conversion_value}

For dynamic CPA campaigns, append:

Dynamic CPA Parameter 
&CPA={CPA_value}
